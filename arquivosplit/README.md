# rust_app
## aplicativos de códigos na linguagem rust

### **Acessar a pasta da aplicação**
```
	$ cd arquivosplit
	$ cargo add toml
	$ cargo add encoding
	$ cargo add encoding_rs
	$ cargo add encoding_rs_io
```
### **Cosntruir a aplicação**
```
	$ cargo build
```
### **Executar a aplicação**
```
	$ cargo run
 ```
