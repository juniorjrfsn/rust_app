# rust_app
## aplicativos de códigos na linguagem rust

### **Acessar a pasta da aplicação**
```
	$ cd mlpmercadofinanc
```
### **Cosntruir a aplicação**
```
	$ cargo build
	$ cargo add rusqlite
```
### **Executar a aplicação**
```
	$ cd mlpmercadofinanc
	$ cargo run
	$ cargo.exe "run", "--package", "mlpmercadofinanc", "--bin", "mlpmercadofinanc"

	$ cargo run --bin mlpmercadofinanc
```
